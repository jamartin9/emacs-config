** swap
- uses hard drive as memory page cache
- ~swapoff /swapfile~ to disable
- ~fallocate -l 2G /swapfile && mkswap /swapfile~ to create
- ~swapon /swapfile~ to enable
  - load zram module and setup swap partition for compression
